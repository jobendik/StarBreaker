# STARBREAKER — Controls / Inputs

The game auto-detects keyboard/mouse and touch. Movement is the only thing you
steer — firing is automatic.

## Desktop (keyboard + mouse)

| Action | Input |
|---|---|
| Move | **W A S D** or **Arrow keys** |
| Dash | **Shift** or **Space** |
| Pause / Resume | **P** or **Esc** |
| Mute / Unmute audio | **M** |
| Menus & upgrade cards | **Mouse click** |
| Fire weapons | **Automatic** (no button) |

## Mobile / touch

| Action | Input |
|---|---|
| Move | **Drag** anywhere on screen (virtual joystick) |
| Dash | Tap the on-screen **⚡** button |
| Pause | Tap the pause control / menu buttons |
| Menus & upgrade cards | **Tap** |
| Fire weapons | **Automatic** (no button) |

## Notes
- The game **auto-pauses** when the tab loses focus or is hidden, and respects
  CrazyGames' audio mute.
- Aiming/targeting is automatic — focus on positioning, dashing out of danger, and
  picking the best upgrade cards.
